Download Navit

Navit is a modular, touch screen friendly navigation system with GPS tracking, realtime routing engine and support for various vector map formats. Works on a wide range of devices, from computers to smartphones.

Windows

If you wish to install Navit on the device, download the .cab file and install it. It is recomended that you download the zip file and copy this onto an SD card instead.
